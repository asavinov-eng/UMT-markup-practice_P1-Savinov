# Flora

Flower shop frontend and backend project.

- Frontend: https://asavinov-eng.github.io/UMT-markup-practice_P1-Savinov/
- Backend API: https://flora-backend-983x.onrender.com/api/bouquets
- Swagger UI: https://flora-backend-983x.onrender.com/api-docs
- Backend repository: https://github.com/asavinov-eng/flora-backend

## Local frontend

Open `index.html` in a browser.

## Local backend

1. Go to `backend`.
2. Copy `.env.example` to `.env` and add `DATABASE_URL`.
3. Run `npm install`.
4. Run `npm start`.
